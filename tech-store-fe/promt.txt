DANH SÁCH API LÀM CĂN CỨ

A) Auth
- POST   /api/auth/register
  Body: { username, email, password }
- POST   /api/auth/login
  Body: { identifier, password }

B) User Management (Admin) – Base: /api/auth/users
- GET    /api/auth/users
- POST   /api/auth/users/add
  Body: { username, email, password, role }
- PUT    /api/auth/users/update?id=<id>
  Body: { username, email, role }
- DELETE /api/auth/users/delete?id=<id>

C) Customers & Loyalty (Admin) – Base: /api/auth/customers
- POST   /api/auth/customers
  Body: { fullName, email, phone, birthDate, customerType, address, notes }
- GET    /api/auth/customers
- GET    /api/auth/customers/{type}   (REGULAR | VIP)
- PUT    /api/auth/customers/{id}
  Body: Partial CustomerRequest
- DELETE /api/auth/customers/{id}

D) Categories (Admin) – Base: /api/categories
- POST   /api/categories
  Body: { name, description, imageUrl, displayOrder, isActive, parent?: { id } }
- GET    /api/categories?activeOnly=true|false

E) Products
- POST   /api/products  (Admin – multipart/form-data)
  Fields: name, sku, description, isVisible, categoryId, imageFile
- GET    /api/products?page=0&categoryId=<id?>
  (Customer + Admin)

F) Orders
- POST   /api/orders (Customer)
  Body: { customerId, paymentMethod, channel, notes, items:[{ variantId, quantity }] }
- GET    /api/orders/new (Admin)
- GET    /api/orders/processing (Admin)
- GET    /api/orders/{orderId}
- PUT    /api/orders/{orderId}
  Body: { paymentMethod, notes }
- PUT    /api/orders/{orderId}/cancel
- DELETE /api/orders/{orderId}

G) Pricing (Admin) – Base: /api/prices
- POST   /api/prices/variants/{variantId}
  Body: { currencyCode, price, costPrice, reason }
- GET    /api/prices/products/{productId}
- PUT    /api/prices/history/{id}
  Body: { currencyCode, price, costPrice, reason }
- DELETE /api/prices/history/{id}
- GET    /api/prices/variants/{variantId}/effective

H) Promotions (Admin) – Base: /api/promotions
- POST   /api/promotions
  Body: { code, name, description, discountType, discountValue, startDate, endDate,
          scope, productIds[], variantIds[], priority, stackable, isActive }
- GET    /api/promotions?activeOnly=true|false
- PUT    /api/promotions/{id}
- DELETE /api/promotions/{id}

I) Settings (Admin) – Base: /api/settings
- GET    /api/settings/currency/default
- PUT    /api/settings/currency/default
  Body: { currencyCode }
